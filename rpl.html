Rekayasa Perangkat Lunak (RPL)

Jurusan Rekayasa Perangkat Lunak (RPL) adalah program keahlian yang fokus pada pembuatan dan pengembangan perangkat lunak berbasis desktop, web, maupun mobile. Siswa akan mempelajari bahasa pemrograman, pengembangan aplikasi, desain antarmuka, serta pengujian dan dokumentasi perangkat lunak secara profesional.

Kenapa Memilih RPL?

  • Belajar langsung membuat aplikasi dan website dari awal.
  • Menguasai berbagai bahasa pemrograman populer seperti HTML, CSS, JavaScript, PHP, dan Java.
  • Berpeluang tinggi bekerja di industri IT atau menjadi freelance developer.
  • Dibimbing oleh guru dan praktisi berpengalaman di bidang teknologi.

Prospek Karier

Lulusan RPL bisa berkarier sebagai programmer, web developer, software engineer, game developer, UI/UX designer, atau melanjutkan pendidikan ke jenjang perguruan tinggi di bidang teknologi.
